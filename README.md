# Analysis of DNA Sequences

Inspired by [University of Alberta, Paul Stothard's Sequence Manipulation Suite](https://www.bioinformatics.org/sms2/index.html), a small bioinformatics library written in C++ to approximately find sequences that can be easily mutated into a useful restriction site.
